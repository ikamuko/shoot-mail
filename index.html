初回セットアップ — 一度設定すれば次回から不要です
1
Gemini APIキーを取得

Google AI Studio でAPIキーを無料で発行できます。
Googleアカウントでログイン → 「Create API Key」をクリック

2
Gmail連携（Google OAuth2）

GmailへのアクセスにGoogle OAuth2クライアントIDが必要です。
Google Cloud Console → 「認証情報を作成」→「OAuthクライアントID」
アプリの種類：ウェブアプリ / 承認済みオリジン：null（ローカルファイルの場合）または実際のURL
※すぐ試したい場合：テスト用として自分のGoogleアカウントでOAuth同意画面を設定してください。

3
追加カテゴリキーワード（任意）

モデル・スタジオ・ヘアメイク・カメラマンは固定で設定済みです。
追加で検索に使いたいキーワードがあればカンマ区切りで入力してください。
例：オーディション, 撮影スペース, フォトグラファー

4
取得期間

直近何日分のメールを取得するか設定します（デフォルト：30日）

新規案件を作成
案件キーワードはメール取得時のAND検索に使われます。
例：キーワードに「春夏」を設定すると 「春夏」AND「モデル」 のようにGmailを検索します。
商品名・企画名・シーズン名など、撮影業務メールに共通して含まれる語句を設定してください。
